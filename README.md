# avariavs-api

###### In terminal 
~~~~
>npm install
>nodemon index.js
~~~~  
###### In Browser
~~~~
localhost:3000
localhost:3000/classes          View JSON object that contains all the classes and nested characters.

localhost:3000/class            Contains all chracters in a class. Replace 'class' with one of the 4 classes.
                                Class: eoni, mage, manasoldier, servo
                                
localhost:3000/class/hero       Contains all chracters in a class. Replace hero with a hero name from the class.
                                Hero: eoni, mage, manasoldier, servo
                                
localhost:3000/class/hero/stat  Contains the value of a given stat.
                                Stat: hp, spd, str, vit, mna, spr
~~~~
